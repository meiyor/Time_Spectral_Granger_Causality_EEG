# Time_Spectral_Granger_Causality_EEG
Granger Causality pipeline for evaluating channels predicitability from data from [Xu et al 2023] described here https://www.pnas.org/doi/abs/10.1073/pnas.2216268120. For downloading the data refer to the Zenodo link here and download the entire [here](https://zenodo.org/record/7803212#.ZC3Cb-zML0q) and download the entire dataset.
